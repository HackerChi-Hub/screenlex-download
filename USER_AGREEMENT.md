# ScreenLex 用户协议

版本：0.1.0
更新日期：2026-06-16

欢迎使用 ScreenLex 光影词库。本软件由黑粉科技 HyphenTech 提供，用于本地影视字幕的英语学习整理。

## 1. 软件用途

ScreenLex 是一款本地桌面学习工具，用于扫描用户本机的影视文件目录，读取用户已有或自行生成的字幕文件，并在本机生成词汇、短语、背景提示、复习记录和双语字幕辅助材料。

本软件不提供影视内容，不内置播放器，不分发电影、剧集、字幕源或任何受版权保护的音视频资源。

## 2. 用户责任

用户应确保导入、扫描、识别、下载、校对、翻译和学习的媒体文件及字幕文件来源合法，并仅在适用法律允许的范围内用于个人学习、研究或合理使用场景。

因用户导入、处理、传播或分享受版权保护内容而产生的法律责任，由用户自行承担。

## 3. 本地数据

ScreenLex 默认在本机处理数据。学习包缓存、字幕处理断点、复习记录和配置文件保存在用户本机应用数据目录中。

用户配置第三方模型、OpenSubtitles 或本地 Whisper 时，相关调用可能会把必要文本发送到用户配置的第三方服务。用户应自行确认对应服务的隐私政策、费用和使用限制。

## 4. 匿名用户统计

为了解软件实际使用规模、版本分布和平台兼容情况，ScreenLex 默认开启匿名用户统计。软件会在后台按天上报匿名安装 ID 的哈希、应用版本、系统平台、CPU 架构和日期。

该统计不会上传用户的视频路径、文件名、字幕内容、词库内容、复习记录、模型 API Key、OpenSubtitles 账号或其它本地学习数据。

用户可在软件设置中通过「用户统计」按钮关闭或重新开启该统计。

## 5. AI 生成内容

AI 精讲、字幕校对、字幕翻译、词汇释义和记忆方法可能存在错误、遗漏或不适合具体语境的情况。软件生成内容仅供学习参考，不构成考试、翻译、法律、医学、财务或其它专业意见。

用户应结合原文、上下文和可靠资料自行判断学习内容的准确性。

## 6. 自动更新

ScreenLex 可通过官方公开下载仓库检查更新。更新包会使用 Tauri updater 签名机制校验，防止安装未经信任的更新包。

请勿从非官方渠道下载或安装 ScreenLex。
